#仿百度搜索加载动画

![](https://github.com/danhanxiang/daidudemo/blob/master/screenshots/GIF2.gif)&nbsp;&nbsp;&nbsp;

##Points

* 中间的圆固定不动，两边的圆对称运动。
* 每次当两边的圆在中点相遇，则改变颜色。
